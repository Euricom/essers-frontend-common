# Pagination

```jsx
import { Pagination } from '@essers/frontend-ui';

export MyComponent = () => (
  const [page, setPage] = useState(0);
  const handlePageChange = (page: number) => {
    setPage(page);
  };
  return (
    <div>
      <Pagination totalItems={100} pageSize={5} page={page} onPageChange={handlePageChange} />
    </div>
  );
);
```

## Properties

The Pagination component is a lightweight wrapper around the [Shadcn/ui - Pagination](https://ui.shadcn.com/docs/components/pagination).
For detailed information about its usage, please refer to the Shadcn/ui documentation.

<ArgTypes of={PaginationStories} />

## Usage

The Pagination component is a easy to use component to render a pagination.
Just provide the total number of items, the page size and the current page.
